# AI Chat Navigator Pro 🚀

这是一个为 ChatGPT, Gemini, Claude 等 AI 工具打造的全局对话侧边导航栏。

## ✨ 核心增强特性 (What I Added)
相比于原版项目，本版本进行了深度重构与增强：
- **🔍 毫秒级全文检索**：新增搜索面板，采用 300ms 防抖处理，支持在隐藏的完整对话文本中进行无感检索。
- **🤖 Claude 深度适配**：重写了 DOM 扫描逻辑，采用多策略回退机制（选择器匹配 + 结构推断），完美兼容 Claude 最新的 Tailwind 前端结构。
- **🚀 突破容量限制**：移除了原版硬编码的 60 条容量上限限制，支持超长对话的完整映射。
- **🎯 焦点控制优化**：解决了原版在输入框激活时意外触发收起动画的交互 Bug。
